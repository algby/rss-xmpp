Simple rss2xmpp bot for Google App Engine
=========================================

Example
-------
rss-xmpp@appspot.com

Bot commands
------------

- HELP [<command>] 
    Show help message for some command.
- PING
    Answer PONG.
- ADD <url>
    Adds a feed url.
- DEL <url>
    Removes a feed from your list.
- FEEDS
    Get list of feeds.
- LOGIN
    Get a key to login via web interface.
